Liunx常用命令（一）

发表于 2016-07-05   |   分类于 liunx   |     |   阅读次数

学习liunx的shell还得从租第一台云服务器开始，那时候只是好奇，觉得要多学点东西，到现在证明学习shell是正确的。在平时我总结了一些自己遇到的比较常用的liunx命令，现在和大家分享一下，以后也会不断更新。

基本命令

  • 返回上一级目录 cd ..
  • 进入指定目录 cd /usr/local
  • 查看目录结构或详细 ls ll
  • 查看当前路径 pwd
  • 创建文件夹 mkdir folder
  • 创建文件 touch file
  • 删除文件 rm -rf path/file
  • 复制文件 cp path1/file1 path2/file2
  • 移动/重命名文件 mv path/file1 path/file2
  • 查看文件 cat file less file more file
  • 打印日志 tail -200f xxx.log
  • 获取指定程序的进程 ps -ef|grep tomcat
  • 查看运行程序的状态 netstat -tunpl
  • 杀死进程 kill -9 pid

编辑文件

输入如上命令后可显示对应的文件内容，当找到需要改动的地方，再输入命令$ i即可开始编辑，当编辑完成，在键盘按“Esc”键，在在终端输入命令 $ :wq保存并退出本次修改，若只输入命令$ q!则只退出不保存本次修改。

小技巧：当需要复制某一行的时候，在没有输入命令$ i的时候，鼠标定位行可以先连续键入 yy,再键入p可完成对鼠标指针所在行的内容。

切换用户

  • su root（对应用户）
  • su 后面不加用户是默认切到 root
  • su 是不改变当前变量
  • su - 是改变为切换到用户的变量
  • 也就是说su只能获得root的执行权限，不能获得环境变量
  • 而su -是切换到root并获得root的环境变量及执行权限

解压命令包

  • tar包
    • 解包：tar xvf FileName.tar
    • 打包：tar cvf FileName.tar DirName
    • （注：tar是打包，不是压缩！）
  • zip包
    • 解压：unzip FileName.zip
  • rar包
    • 解压：rar x FileName.rar
    • 压缩：rar a FileName.rar DirName
  • gz包
    • 解压1：gunzip FileName.gz
    • 解压2：gzip -d FileName.gz
    • 压缩：gzip FileName
  • tar.gz包 和 tgz包
    • 解压：tar zxvf FileName.tar.gz
    • 压缩：tar zcvf FileName.tar.gz DirName
  • rpm包
    • 解包：rpm2cpio FileName.rpm | cpio -div

Linux下查看、关闭及开启防火墙命令

  1. 永久性生效，重启后不会复原
    开启： chkconfig iptables on 关闭： chkconfig iptables off
  2. 即时生效，重启后复原
    开启： service iptables start 关闭： service iptables stop
    需要说明的是对于Linux下的其它服务都可以用以上命令执行开启和关闭操作。
    在开启了防火墙时，做如下设置，开启相关端口， 修改/etc/sysconfig/iptables 文件，添加以下内容：
    -A RH-Firewall-1-INPUT -m state --state NEW -m tcp -p tcp --dport 80 -j ACCEPT
    -A RH-Firewall-1-INPUT -m state --state NEW -m tcp -p tcp --dport 22 -j ACCEPT
  3. 查看防火墙状态
    chkconfig iptables --list
