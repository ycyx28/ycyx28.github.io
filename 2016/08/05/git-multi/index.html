git多账号

越来越多的企业和开发者将项目托管在git上，有自己搭建的git服务器，也有直接托管在互联网，可能很多人有多个git账户，习惯用命令操作的程序员就有烦恼了，多个git用户怎么解决，博主就有3个git账号，该如何无缝切换？带着这个问题我们往下看。

  • git clone 项目时提示Host key verification failed
    使用 ssh 协议时会验证对方服务器的key。ssh无法确认服务器出示的key 是可信认的，所以问你这个key是不是真的是你要连接的那个服务器的。你没说「yes」所以 ssh 认为你不想继续连接，结果当然是连接失败

  • git clone Permissions 0644 for ‘~/.ssh/id_rsa’ are too open
    只要把权限降到0600就可以了
  • Bad owner or permissions on /home/xxx/.ssh/config
    ssh config文件权限的问题,把权限降到0600就可以了.http://my.oschina.net/meilihao/blog/157716 字数：580
